Social media share and bookmark buttons with integrated analytics. Automatically display buttons for those social networks, where visitor registered.

== Description ==

LessButtons automatically display sharing buttons for those social networks, where visitor registered and logged in.

Support automated detection and display buttons for Facebook, Twitter, Google+, Pinterest, LinkedIn, Reddit, VKontakte, Odnoklassniki.
100+ more social networks and bookmark services available to users.

= Features =

* Buttons for those social networks, where visitor registered.
* Dynamically updatable share counters on all buttons.
* High-resolution icons for retina displays.
* Fully customizable appearance.
* HTTPS protocol for secure communication.
* Tracking social network detections and shares in Google Analytics.
* Email sharing via Gmail, Yahoo! Mail, Outlook.com (Hotmail), AOL mail or standalone apps.

= Statistics =

If you have Google Analytics tracking code on pages, it will automatically receive tracking notifications for each visitor registered in social network or share button pressed. It give you know what social media is popular with your visitors.

= Find Shares =

Magnifier sign on Twitter, Google+ and VK buttons allows your users to find more peoples who share your page.

= Custom Placements =

* Before content, after content.
* Left and right side floating share bars.
* Horizontal floating share bar on bottom of window.
* Special place option for mobile devices.

== Installation ==

= Automatic installation through WordPress =

1. Log-in to your WordPress site.
2. Hover over Plugins and click Add New.
3. Under Search type in "lessbuttons", then click Search Plugins.
4. In the results page, click Install Now.
5. Once installed, click Activate Plugin. You're done!

= Manual installation =

1. Download fresh version here [https://wordpress.org/plugins/lessbuttons/](https://wordpress.org/plugins/lessbuttons/) .
2. Unzip the lessbuttons.*.*.zip file. 
3. Upload the lessbuttons folder (not just the files in it!) in your /wp-content/plugins folder. If you're using FTP, use 'binary' mode.
4. In your WordPress administration, go to the Plugins page.
5. Activate the LessButtons plugin and a subpage for LessButtons will appear in your Options menu.

To upload the plugin through WordPress, instead of FTP - upload the downloaded zip file on the 'Add New' plugins screen in your WordPress admin area and activate.

== Frequently Asked Questions ==

= How to change left side floating share bar top position? =
use style: .lessbuttons_com_slider { top: 150px !important; }
"style" tag with this code can be inserted into "text" widget.

= How to move bottom share bar to right side? =
use style: .lessbuttons_com_slider_internal { position: fixed !important; right: 0px !important; bottom: 0px !important; }
